Engine SSH Credentials

Auto-enrollment via SMC API. FlexEdgeAdmin uses the SMC's change_ssh_pwd endpoint to set a 64-char random root password on each node, then logs in over SSH (with a pinned host fingerprint) to read DHCP lease files and write reservations. You never type or see the password. Connection over the network requires an explicit allow rule — installed in the engine's policy, visible in Management Client, removed on credential delete.
1. FEA source IP per tenant

The IP address engines should accept SSH connections from. Used as the source in the SSH allow rule we install. If FlexEdgeAdmin is behind NAT, this is the egress IP the engine will see — usually a public IP, but can be a private LAN address if FEA and the engine are on the same network.
